# APRI-Based Liver Risk Assessment System

A web-based healthcare application developed during Practice School at Yashoda Hospitals for non-invasive liver fibrosis risk assessment using APRI scoring.

---

## Features

- APRI score calculation
- Liver fibrosis risk analysis
- Admin login authentication
- Patient data visualization
- Graphical analytics dashboard
- Clinical feedback integration

---

## Tech Stack

- HTML

- JavaScript
- Google Apps Script
- Google Sheets
